Looping-Louie-Project
=====================

Ziel ist es, eine möglichst spannende und spaßige Ansteuerung für Looping Loui zu schaffen.


Beschreibung der Ordnerstruktur und Inhalte:
--------------------------------------------
Arduino         //Ansteuerung über einen Arduino
   - only       //Ansteuerung nur über den Arduino, keine Kommunikation nach außen
      - random  //zufällige Geschwindigkeitsänderung 
      - Sinus   //sinusförmige Geschwindigkeitsänderung
   - Komm       //Mit Kommunikation nach außen, Auswahlmöglichkeit der Modi über ein GUI 
      - Arduino //entsprechender Arduinocode
      - GUI     //entsprechender Code für das GUI

Bluetooth              //möglichst platzsparende Hardeware zu und eine möglichst mobile 
			 Kommunikation (Bluetooth, Android App) zu ermöglichen.
   - App	       //Sourcecode App	
   - Microcontroller   //Sourcecode Microcontroller
   	

technische Umsetzung   //Erklärung zur genauen technischen Umsetzung
changelog              //Was wann geändert wurde
